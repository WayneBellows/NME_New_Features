Intune Cross-Tenant Support

Link, view, and manage additional Intune tenants from a single Nerdio Manager instance. Perfect for MSPs and multi-tenant enterprises.

  • Single pane of glass for all tenants
  • View devices across identity tenants
  • Unified endpoint management
  • Simplified multi-org operations
New Capability

Physical Endpoints now in Real-Time Insights

Intune-managed Windows physical endpoints are now supported with full Real-Time Insights monitoring and alerting.

20K endpoints supported
@ 2-minute polling
  • Last-mile network diagnostics
  • Host, Ethernet, Gateway & Internet health
  • Real-time performance metrics
New Feature

Migrate directly to Windows 365

Enable direct migration of Azure VMs to Windows 365 Enterprise Cloud PCs. No intermediate image capture steps required.

  • Direct AVD Personal → W365 migration
  • Citrix & Omnissa to Cloud PC
  • Simplified provisioning policy setup
  • User settings preserved
New Feature

Multi-Region High Availability

Generate infrastructure-as-code scripts for deploying HA setups across multiple Azure regions. Find it under Settings > Environment > Nerdio > Nerdio Manager Resilience.

New Option

Sysprep in User Context

Run sysprep in either System or User context during desktop image creation. Provides flexibility for different imaging scenarios and application requirements.

  • System-context (default)
  • User-context for special apps
  • Timezone redirection support
  • More control over imaging process
New Option

Deploy to Clone or Source

When using "Set as image," choose to deploy applications and scripted actions to either the source image or the cloned image.

  • Preserve source image integrity
  • Test on clones before production
  • Flexible deployment workflows
  • Reduced image corruption risk
New Capability

Clone Scripted Sequences

Sequences, task groups, and tasks can now be cloned for faster iteration and testing. Plus, automatically sign PowerShell scripts delivered via Intune.

  • One-click sequence cloning
  • Clone task groups & individual tasks
  • Auto-sign Intune PowerShell scripts
  • Use trusted certificates for signing
